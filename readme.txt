Display the Wadvisor Page and Question Widgets.

== Description ==
This Wordpress plugin help you to integrate Wadvisor (https://wadvisor.com) Page and Question Widgets to your Wordpress.

Please visit http://blog.wadvisor.com/tag/widget/ for more details

== Installation ==

= From your WordPress dashboard =

1. Click `Add New` from the plugins page in your wordpress site
2. Search for `Wadvisor Plugin`
3. Click on install

= Alternatively from wordpress.org =

1. Download the latest version of Wadvisor Plugin
2. Extract the files
3. Upload the entire `wp-wadvisor` folder to the `/wp-content/plugins/` directory.

4. Activate the plugin through the 'Plugins' menu in WordPress.
